About Me

Getting Started

I delved into web development when I wanted to create a website for my friend's photography. Soon enough I was an intern for a printing company learning HTML & CSS, relying heavily on tables to create my first website. The cycle of reading and practicing new techniques would continue and soon enough, I was making some pretty damn good looking pages (without tables).

Today

I continue to improve on my skills as a coder. During/after work I am looking for ways to improve my workflow and create clean, efficient code. Not only does it make me a better coder, but it also makes my company's code better as well. There is always something new to learn and why I love being a Web Developer playing catch-up.
